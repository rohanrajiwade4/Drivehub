Welcome To Car Rental Portal

This Car Rental Portal Management System is designed for a web services provider where the user can visit and book their required Car online and select their Car, duration option etc. This system include a complete Car Rental Portal management system so that the admin can easily add, update, or delete any car, check users detail etc. This project mainly purpose will increase clients because customers who live far from showroom can also approach and car can be delivered at his/her doorstep. In this Project used Technology for frontend HTML5, CSS3, JavaScript, Bootstrap and Backend used are Python Django & SQLite.
